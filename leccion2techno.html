1. What is the Internet?

The Internet is a global network that connects millions of computers and devices worldwide, allowing information sharing and communication.

Reflection activity

Think about the ways you use the Internet daily. Write down three activities you do online.

2. Understanding Web Browsers

Web browsers let you access and view websites by interpreting the code behind them.

Try This

List the web browsers you have used. Which one do you prefer and why?

3. What are Search Engines?

Search engines help you find information on the Internet quickly by indexing websites and showing relevant results.

Try This

Write down a question you have and try finding the answer using a search engine. Compare results.

4. Using Email

Email is a quick way to send messages, documents, and media electronically to others anywhere in the world.

Try This

Create a sample email (without sending) and practice formatting a subject line, message, and attachment.
